# Netflix Data Analysis Project

## Overview
This project performs exploratory data analysis on the Netflix dataset using Python.  
The objective is to analyze content distribution, growth trends, genre popularity and country wise production patterns through meaningful visualizations.

This project demonstrates practical use of Pandas for data manipulation and Matplotlib for data visualization.

---

## Dataset
- **Source:** Netflix Titles Dataset
- **Format:** CSV
- **Columns Used:**
  - show_id
  - type
  - title
  - country
  - date_added
  - release_year
  - rating
  - duration
  - listed_in

---

## Tools and Libraries
- Python 3
- Pandas
- Matplotlib

## Data Cleaning Steps
- Converted `date_added` to datetime format
- Extracted year from `date_added`
- Handled missing values in country and genre columns
- Split multi value fields such as country and genres for accurate analysis

---

## Visualizations Created

### 1. Movies vs TV Shows Distribution
Shows the overall distribution of Movies and TV Shows available on Netflix.

### 2. Content Added Over Years
Analyzes how Netflix content has grown over time.

### 3. Movies vs TV Shows Growth Trend
Compares yearly growth patterns between Movies and TV Shows.

### 4. Top 10 Genres on Netflix
Identifies the most popular genres available on the platform.

### 5. Top Content Producing Countries
Highlights countries contributing the highest number of titles.

### 6. Movies vs TV Shows by Top Countries
Compares Movies and TV Shows production across the top content producing countries.

---

## Key Insights
- Netflix has steadily increased its content library over the years
- TV Shows have shown significant growth compared to Movies in recent years
- Drama and International content dominate the platform
- United States and India are among the top content producing countries
